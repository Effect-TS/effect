# @effect/test

Shared test utilities for Effect test runner integrations.

This package provides the core infrastructure used by `@effect/vitest` and `@effect/bun-test`.

## Installation

This package is typically not used directly. Instead, use one of the test runner-specific packages:

- `@effect/vitest` - For Vitest
- `@effect/bun-test` - For Bun's test runner
