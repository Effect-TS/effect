# combinators

## forEach

Applies the specified function to each element of the `List`.

**Signature**

```ts
export declare const forEach: {
  <A, B>(f: (a: A) => B): (self: Chunk<A>) => void
  <A, B>(self: Chunk<A>, f: (a: A) => B): void
}
```
